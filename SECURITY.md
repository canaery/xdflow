# Security Policy

## Supported Versions

We support the latest minor release on the `main` branch. Security fixes are typically released in patch versions.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it privately:

- Email: `julien@canaery.com`

Please include a detailed description, steps to reproduce, and any relevant logs. We will acknowledge receipt within
7 days and provide a timeline for remediation when possible.
